# 💝 我们的小确幸 - 情绪盲盒互动游戏

> 一个记录感情点滴的互动网页小游戏

## 📖 项目简介

这是一个温馨有趣的网页互动游戏，用来记录和回顾恋爱中的各种时刻。通过"回答问题 → 抽取卡片"的方式，既能记录当下的心情，又能随机看到过去的美好回忆。

## ✨ 核心功能

### 1. 问答环节（入口）
- 每次进入游戏前，需要回答一个随机问题
- 问题涉及当下心情、回忆、未来期待等
- **字数限制**：必须达到 100 字才能提交
- 实时显示字数统计
- 所有回答自动保存到本地，可随时回看

### 2. 情绪盲盒抽卡（核心玩法）
三个不同类别的盲盒，每个代表不同的情绪类别：

#### 🎁 开心时刻（粉色盲盒）
记录让人感到温暖和快乐的瞬间：
- 党岭聊天化解不快
- 机场接机的温暖
- 工作日带花的惊喜
- 加班时的陪伴与安心
- 等等...

#### 💙 小摩擦（蓝色盲盒）
记录相处中的磕磕碰碰：
- 周末计划不确定的不开心
- 国庆安排的优先级差异
- 帮忙做决定的介意
- 抽烟频率的担心
- 等等...

#### ✨ 新发现（紫色盲盒）
记录对彼此的新认识和发现：
- 恋家程度的差异
- 工作变动的担心
- 爱好偏好的不同
- 注意力分配的规律
- 颜控的自我发现
- 等等...

### 3. 卡片展示
- 3D翻转动画效果
- 柔和的配色方案（莫兰迪色系）
- 点击盲盒，随机抽取该类别的一条内容
- 已抽取进度显示（如：15/22）

### 4. 回答记录本
- 查看所有历史回答
- 按时间排序
- 支持导出功能（可选）
- 本地存储，隐私安全

## 🎨 设计方案

### 页面结构
```
首页（问答页）
    ↓
抽卡页（三个盲盒）
    ↓
卡片展示（翻转动画）
    ↓
继续抽取 / 查看记录
```

### 视觉风格
- **配色**：温柔的莫兰迪色系
  - 粉色：开心时刻 (#FFE4E1)
  - 蓝色：小摩擦 (#B0C4DE)
  - 紫色：新发现 (#E6E6FA)
- **字体**：优雅易读的中文字体
- **动画**：流畅的卡片翻转、粒子效果
- **背景**：简洁温馨

### 交互设计
1. **输入框**：
   - 实时字数统计
   - 未达100字时按钮禁用且灰色
   - 达到100字按钮高亮闪烁

2. **盲盒**：
   - hover 时放大效果
   - 点击后盲盒打开动画
   - 卡片从盲盒中"飞出"

3. **卡片**：
   - 3D翻转效果
   - 背景模糊突出卡片
   - 点击空白处关闭

## 🛠 技术栈

### 核心技术
- **HTML5** - 页面结构
- **CSS3** - 样式和动画（transform, transition, animation）
- **Vanilla JavaScript** - 交互逻辑（无需框架，保持简单）

### 数据存储
- **localStorage** - 本地存储回答记录和抽卡进度
- JSON 格式存储数据

### 可选增强
- **anime.js** 或 **GSAP** - 更丰富的动画效果
- **Canvas** - 粒子特效（爱心飘落等）

## 📝 问题库设计

### 关于当下（40%）
- 今天有什么让你开心或难过的事情吗？
- 最近最想和他分享什么？
- 此刻的心情如何？用三个词形容一下
- 今天最感激他的一件事是什么？

### 关于回忆（30%）
- 最近有什么新的小确幸吗？
- 有没有新发现他什么有趣的小习惯？
- 你们之间有什么新的"默契时刻"？
- 最近哪个时刻让你感觉特别被爱？

### 关于未来（20%）
- 下次约会最想去哪里/做什么？
- 有什么想一起实现的小目标？
- 想对一个月后的自己说什么？
- 最期待和他一起做的事情是什么？

### 关于感悟（10%）
- 最近对这段感情有什么新的感悟？
- 你觉得你们相处模式有什么变化？
- 现在的相处状态满意吗？为什么？

## 💖 使用说明

1. 打开 `index.html`
2. 阅读随机问题，输入至少 100 字的回答
3. 提交后进入抽卡页面
4. 选择一个盲盒类别点击
5. 查看抽到的卡片内容
6. 可继续抽取或查看历史记录

## 🔐 隐私说明

- 所有数据仅存储在本地浏览器（localStorage）
- 不会上传到任何服务器
- 清除浏览器数据会清空记录
- 建议定期导出备份

## 📌 注意事项

1. 建议使用现代浏览器（Chrome、Firefox、Safari、Edge）
2. 首次使用需要初始化数据
3. 可以随时重置进度重新开始
4. 导出的记录可以保存为纪念
